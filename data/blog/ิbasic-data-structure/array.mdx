## โครงสร้างข้อมูลประเภท Array

อาร์เรย์(Array) เป็นโครงสร้างข้อมูลที่นำเซลล์ข้อมูล (Data cell) มาเรียงต่อกันเป็นกลุ่มข้อมูล โดยข้อมูลแต่ละเซลล์ในอาร์เรย์จะมีชนิดข้อมูลพื้นฐาน(Integer, float, double, ...) เป็นชนิดข้อมูลเดียวกันทั้งกลุ่ม และ ต้องกำหนดขนาดของอาร์เรย์หรือจำนวนเซลล์ข้อมูลที่จะจัดเก็บก่อนที่จะนำไปใช้งาน

การเข้าถึงตำแหน่งเซลล์ข้อมูลในอาร์เรย์จะใช้ชื่ออาร์เรย์และตำแหน่ง... -> comming soon
