## What is Rust?

Rust เป็นภาษาโปรแกรมภที่พัฒนาโดย Mozilla โดยมุ่งเป้าว่าต้องการเป็นภาษาสำหรับงานประมวลผลที่ทำงานพร้อมๆ กัน (concurrency) และทำงานบนระบบขนาดใหญ่ ออกแบบมาไม่ให้มีปัญหาด้านหน่วยความจำอย่าง buffer overflow

และในภาษานี้เรื่องที่ ใครๆ ก็ต้องพูดถึงก็คือ ในเรื่องของการจัดการ Memory ไม่ว่าจะเป็นด้านความปลอดภัยหรือด้าน Performance ที่ดีเยี่ยมและ สามารถทำงานร่วม Embedded Device ได้

## Rust ทำอะไรได้บ้าง

Rust สามารถนำไปประยุกต์ใช้ได้หลายหลาย ขึ้นอยู่กับความต้องการของเรา เช่น

- Command-line App
- WebAssembly
- Networking
- Embedded device

## Installation

เรามาติดตั้ง Rust กันดีกว่า
ติดตั้งแบบไฟล์ .exe สำหรับ Windows --> [Download](https://www.rust-lang.org/tools/install)

สำหรับคนที่ใช้ macOS, Linux ให้รันในคำสั่ง Terminal ได้เลย

```
curl https://sh.rustup.rs -sSf | sh
```

สำหรับคนที่ใช้ Windows Subsystem for Linux

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## ลองเขียนโค้ดกัน

แน่นอนครับว่าโปรแกรมแกรมแรกที่เราจะเขียนทุกครั้งเราเริ่มเรียนภาษาใหม่ๆ นั้นคือโปรแกรมแสดงข้อความ "Hello, World!" ออกทาง Command Line นั้นเอง

ขั้นตอนแรกให้เราสร้างไฟล์ที่มีชื่อ `hello_world.rs` ซึ่งใน rust จะใช้นามสกุลไฟล์เป็น `.rs` นะครับ

```rust:hello_world.rs showLineNumbers
fn main() {
    println!("Hello, World!");
}
```

ในโปรแกรมของเรามีการประกาศฟังก์ชั่น _main_ และภายในมีการใช้งานคำสั่ง `println!()` ซึ่งเป็นคำสั่งแสดงผลออกทาง Command Line โดยมีใส่ข้อความ "Hello, World!" และจบคำสั่งด้วยการใส่เครื่องหมาย Semicolon(;)

ต่อมา มาที่ Terminal หรือ CMD เช็คดู Path ที่อยู่ของโปรแกรมให้ดีๆ

เราจะลอง Compile โปรแกรมของเรากันโดยใช้คำสั่ง

```
rustc hello_world.rs
```

ทีนี้เราจะมาลอง Run โปรแกรมเราโดยใช้คำสั่ง

```
./hello_world
```

Output:

```
Hello, World!
```
